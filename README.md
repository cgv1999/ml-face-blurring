# ML System for Face Blurring

Воспроизводимый ML-пайплайн для автоматического размытия лиц на видео в соответствии с требованиями КоАП РФ Статья 13.11.3.

## Структура проекта
├── HW5_MLOps_Чернигов_Григорий.ipynb # Ноутбук с домашним заданием!!! 
├── README.md # Документация
├── dvc.yaml # DVC пайплайн
├── params.yaml # Гиперпараметры
├── requirements.txt # Зависимости
├── ml_system_for_face_blurring.png # Архитектурная схема
└── src/
├── prepare.py # Подготовка данных
├── train.py # Обучение модели
└── blur_faces.py # Размытие лиц

## Инструменты
- **DVC** — версионирование данных и воспроизводимость
- **MLflow** — отслеживание экспериментов
- **Feast** — хранилище признаков
- **Marimo** — реактивные ноутбуки
- **Diagrams** — архитектурная схема
- **OpenCV** — обработка видео
